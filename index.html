FAQ

Čo je Download a Upload?
Download je rýchlosť sťahovania (napr. videá, weby). Upload je rýchlosť odosielania (napr. nahrávanie súborov, videohovory).
Čo je Latency (ping)?
Čas (ms), za aký paket dorazí na server a späť. Nižšie = lepšia odozva (hraní, hovory).
Čo je Jitter?
Kolísanie latencie v čase. Nižší jitter = stabilnejšie spojenie (hlas/video).
Čo je Packet loss?
Percento stratených paketov. Vyššia strata = sekanie hovoru, lagy, výpadky.
Ako sa ráta Score (Excellent/Good/Bad)?
Score je zhrnutie kvality podľa meraní + nastavení (frekvencia, trvanie, packet size, acceptable delay). Excellent je najlepšie.

Ako to funguje?

1) Spustíš test
Klikni Start Test. Aplikácia postupne simuluje: latency → jitter → packet loss → download → upload.
2) Dostaneš výsledky
Uvidíš čísla + grafy. Graf je zelený, ak je hodnota v norme, červený, ak je mimo.
3) Score
Excellent (zelené), Good (čierne), Bad (červené) – zhrnutie kvality spojenia.
4) History
Test sa automaticky uloží do histórie. Po rozkliknutí vidíš aj grafy.
Čo znamenajú metriky?
Latency = odozva v ms (nižšie lepšie), Jitter = kolísanie odozvy (nižšie lepšie), Download/Upload = rýchlosť (vyššie lepšie), Packet loss = strata paketov (nižšie lepšie).
